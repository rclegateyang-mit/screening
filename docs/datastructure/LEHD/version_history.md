## 8. Release History and Errata

### 8.1. Snapshot S2021_R2022Q4

- Updated with all available job data through 2022Q1
- Major differences from 2014 LEHD Snapshot:
  - Removed ICF_US_ADDRESSES table; residence information now appears on ICF_US_RESIDENCE_CPR and ICF_US_RESIDENCE_RCF
  - Removed GAL tables; establishment location provided on ECF_ZZ_SEINUNIT
  - Removed U2W tables; establishment imputes provided on JHF_ZZ
  - Assorted structural changes to data tables
- Updated and revised documentation

### 8.2. Snapshot S2022_R2023Q4

- Updated with all available job data through 2023Q1
- Added 2022 NAICS codes

### 8.3. Snapshot S2023_R2024Q4

- Updated with all available job data through 2024Q1
- Latest Census geography contains all new county codes for Connecticut

### 8.4. Snapshot S2024_R2025Q4

- Updated with all available job data through 2025Q1
- Added Parquet output file format
