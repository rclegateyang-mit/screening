# LEHD Snapshot Documentation - Employer-Level Files

Source URL: https://lehd.ces.census.gov/data/lehd-snapshot-doc/S2024/sections/employer_level.html

## Page Overview

This page is Section 3 of the LEHD Snapshot Documentation (Release S2024_R2025Q4) from the U.S. Census Bureau, focusing on employer-level data files.

## Main Content Structure

The documentation covers three primary employer-level file types:

### 3.1 Employer Characteristics File (ECF)
Includes subsections on:
- Overview
- User Guidance
- Codebook documentation for three file variants: SEINUNIT, SEIN, and T26 files

### 3.2 Successor-Predecessor File (SPF)
Organized into:
- Overview section
- User Guidance
- Codebook details

### 3.3 Quarterly Workforce Indicators (QWI)
Contains:
- Overview
- User Guidance
- Codebook specifications

The page includes JavaScript functionality for version selection, allowing users to switch between different documentation releases via a dropdown menu. A CSV file manages available documentation versions.
