# LEHD Snapshot Documentation - Selected Codebook Value Labels

Source URL: https://lehd.ces.census.gov/data/lehd-snapshot-doc/S2024/appendices/value_labels.html

## Document Overview

This documentation section provides categorical variable labels used across LEHD snapshot tables. The page contains dictionary identifiers (_dict_id_) for CSV downloads and comprehensive code-to-label mappings.

## Geography Codes

### State Codes (Section 6.1.1)
The state coding system uses two-digit numeric identifiers. Examples include Alabama (01), Alaska (02), Arizona (04), and extends through Wyoming (56). Puerto Rico is assigned code 72. A CSV download is available for all state codes.

### CBSA Codes (Section 6.1.2)
Core-based statistical areas receive five-digit codes. The listing ranges from "Aberdeen, SD" (10100) through "Zanesville, OH" (49820), with "Nonmetropolitan, nonmicropolitan, AL" assigned code 99999. Over 900 geographic areas are documented.

### County Codes (Section 6.1.3)
County identification uses five-digit codes structured as state FIPS code plus county number. For instance, Alabama counties begin with 01 (01001 for Autauga through 01133 for Winston). All U.S. states and territories are represented, including Puerto Rico municipalities.

### Workforce Investment Board Codes (Section 6.1.4)
WIB codes identify regional labor market areas. These alphanumeric identifiers organize workforce development regions across states, such as "North Alabama Works" (01010001) and "Eastern Connecticut WIA" (09090001).

## Industry Classification Systems

The documentation references multiple NAICS versions (2022, 2017, 2012, 2007, 2002, 1997) and SIC 1987 codes, though detailed tables appear truncated in the provided content.

## Temporal Variables

Section 6.3.1 references "Indexed Quarters (qtime)" for temporal data organization in LEHD datasets.
