# LEHD Snapshot Documentation: Person-Level Files

Source URL: https://lehd.ces.census.gov/data/lehd-snapshot-doc/S2024/sections/person_level.html

## Document Overview

This page serves as the table of contents for Section 4 of the LEHD Snapshot Documentation (Release S2024_R2025Q4), focusing on person-level data files from the U.S. Census Bureau.

## Main Content Structure

The page organizes person-level files into two primary categories:

### 4.1 Individual Characteristics Files - Demographics
This section covers demographic information with subsections including:
- Overview
- User guidance materials
- Codebook for the ICF_US file
- Codebook for ICF_US_IMPLICATES_AGE_SEX_POB file
- Codebook for ICF_US_IMPLICATES_EDUCATION file
- Codebook for ICF_US_IMPLICATES_RACE_ETHNICITY file

### 4.2 Individual Characteristics Files - Residential Geography
This section addresses geographic residence data with:
- Overview
- User guidance
- Codebook for ICF_US_RESIDENCE_CPR file
- Codebook for ICF_US_RESIDENCE_RCF file

## Additional Documentation Sections

The broader documentation includes sections on jobs-level files, employer-level files, acronyms, codebook value labels, bibliography, and release history.
